MDK CLI Templates Repository
==================================

This repository holds every template used by MDK CLI. Each branch hold one
specific template. There are currently 4 branches :

 * __basic__ : A Basic MDK Template
 * __design__ : The Basic MDK template with a design overlay : allows to natively
				generate nicer applications by overriding styles and resources
 * __my\_expenses\_demo__ : A demo template based on __design__ and bringing the
							my Expenses application model to the created project.
							Project is buildable out-of-the-box.
 * __my\_expenses\_tuto__ : A tutorial template based on My Expenses application.


To retrieve a project, simply checkout the relevant branch and start working.
The _master_ branch is designed to be left empty.

__*Warning:* Branches are not designed to ever be merged !__
