MDK CLI Templates Repository

This repository holds every template used by MDK CLI. Each branch hold one specific template. There are currently 4 branches :

  • basic : A Basic MDK Template
  • design : The Basic MDK template with a design overlay : allows to natively generate nicer applications by overriding styles and resources
  • my_expenses_demo : A demo template based on design and bringing the my Expenses application model to the created project. Project is buildable out-of-the-box.
  • my_expenses_tuto : A tutorial template based on My Expenses application.

To retrieve a project, simply checkout the relevant branch and start working. The master branch is designed to be left empty.

Warning: Branches are not designed to ever be merged !
